# 🚗 Car Sales Data Analysis & Cleaning Project

## 📌 Overview

This project focuses on cleaning, transforming, and analyzing a car sales dataset to generate actionable business insights for decision-making. The goal is to convert raw, inconsistent data into a structured, analytics-ready dataset that supports reporting and executive dashboards.

---

## 🛠️ Tools & Technologies

* **Databricks / Spark SQL** – Data cleaning and transformation
* **SQL** – Data manipulation and feature engineering
* **Power BI / Looker Studio** – Data visualization and dashboarding
* **Excel / PowerPoint** – Reporting and presentation

---

## 🔄 Process

1. **Data Cleaning**

   * Removed duplicates using window functions
   * Standardized text fields (make, model, transmission, etc.)
   * Converted state abbreviations to full names
   * Handled missing values using imputation techniques
   * Fixed date formats and numeric inconsistencies



2. **Data Transformation**

   * Created derived fields (Age groups, pricing categories, flags)
   * Built data quality indicators (outliers, missing values)
   * Structured dataset for analytics use

3. **Data Analysis**

   * Revenue and sales trend analysis
   * Vehicle performance by category
   * Pricing comparison (Selling Price vs MMR)
   * Geographic sales insights
   * Identification of high-value transactions

---

## 📊 Key Insights

* Revenue is driven by a small number of high-performing vehicle categories
* Pricing inconsistencies present both risks and margin opportunities
* Certain regions significantly outperform others
* Data quality issues (duplicates, missing values) impact reporting accuracy

---

## 🚀 Outcome

The project delivers a **clean, reliable dataset** and **executive-level insights** that support:

* Better pricing decisions
* Improved sales strategy
* Enhanced data quality and reporting
